# Security Policy

## Supported Versions

This project is a v0.1.0 public skill package. Security fixes are made on the
default branch until a stable release line exists.

## Reporting a Vulnerability

Please report suspected vulnerabilities through GitHub private vulnerability
reporting when available, or by opening a minimal public issue that avoids
sharing private resumes, recruiter messages, credentials, tokens, or live job
application data.

Include the affected version or commit, reproduction steps using synthetic data,
and the expected impact. Do not include real candidate contact details or
private employer communications in reports.

## Data Handling

The skill is designed for local, evidence-backed job-search workflows. It should
not commit private resumes, job-board credentials, recruiter messages, salary
history, work-authorization details, or other personal application material to
this repository.
